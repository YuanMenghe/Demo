# 若生 NOAH AI 科研平台 Demo

This repository contains the local demo for the Ruosheng / NOAH AI research workflow platform.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Optional: set `VITE_THESIS_URL` in `.env.local` if the narrative review workflow needs an authorized URL.
3. Run the app:
   `npm run dev`
