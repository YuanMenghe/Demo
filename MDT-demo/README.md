# 智汇MDT - 复杂病例智能体平台

一个用于MDT（多学科会诊）病例管理的交互示意网站。

## 功能特点

### 1. 患者管理界面（主页面）
- 患者卡片列表，支持卡片/列表视图切换
- 科室分组管理（肝癌组、淋巴瘤组、肺癌组、胃癌组等）
- 实时搜索过滤（支持姓名+疾病、合并症等关键词）
- 拖拽患者到分组
- 脱敏显示患者信息

### 2. 添加/补充资料界面（弹窗）
- 拖拽上传医疗文档（支持PDF、DOC、DOCX、PPT、PPTX、JPG、PNG）
- 患者信息文本输入
- AI智能解析提示
- 生成患者信息按钮

### 3. MDT病例信息看板（核心界面）
- 患者基本信息展示（姓名、性别、年龄、主疾病、分期）
- 诊疗时间旅程
  - 横向时间轴视图
  - S型时间轴视图
  - 可点击节点查看详情
- 关键指标趋势图
  - 支持多疾病类型指标（肝癌、肺癌、淋巴瘤、胃癌）
  - 可切换显示不同指标
  - Chart.js绘制的趋势图
- 患者基本信息摘要
- 补充资料/下载报告功能

### 4. 病例详情页面
- 左侧事件导航列表
- 诊疗事件详情
- 关键指标状态（正常/异常标识）
- 实验室检查结果表格

### 5. 账号管理界面
- 统计面板（总病例数、活跃科室、AI解析成功率、在线医生）
- 科室权限管理卡片
- 各科室支持病种配置

### 6. 设置界面
- 隐私与脱敏设置
  - 脱敏处理方式选择
  - 隐私关键词配置
  - 数据存储位置选择
- AI模型设置
  - 解析模型选择
  - 处理超时时间配置
- 操作日志查看与筛选

### 7. 交互特效
- AI处理动画（文档OCR识别 → 隐私信息脱敏 → 结构化信息提取 → 生成患者画像）
- 弹窗动画效果
- 拖拽上传动画
- Toast提示
- 平滑过渡动画

## 技术栈

- 原生HTML5 + CSS3 + JavaScript
- Font Awesome图标库
- Chart.js图表库

## 使用方法

1. 直接用浏览器打开 `index.html` 文件
2. 或使用本地服务器运行：
   ```bash
   npx serve .
   ```
   然后访问 http://localhost:3000

## 模拟数据说明

项目包含8个模拟患者数据，涵盖：
- 肝细胞癌（2例）
- 弥漫大B细胞淋巴瘤（1例）
- 霍奇金淋巴瘤（1例）
- 肺腺癌（1例）
- 肺鳞癌（1例）
- 胃癌（2例）

每个患者包含完整的：
- 基本信息
- 诊疗时间线
- 检验指标趋势
- 主诉、现病史、既往史、家族史等

## 设计原则

1. **以医生工作流为核心**：贴合MDT会诊前资料整理、会诊中讨论、会诊后总结的流程
2. **数据结构化优先**：自动解析多源数据，将非结构化信息转化为结构化患者画像
3. **隐私安全第一**：本地脱敏处理，确保医疗敏感信息不外泄
4. **可视化辅助决策**：通过时间旅程、指标趋势图等可视化工具辅助决策

## 扩展性设计（II期规划）

- 循证医学支持：基于患者情况检索PubMed/指南
- PPT生成：自动将时间旅程、指标图转化为查房幻灯片
- 临床研究匹配：输入患者信息，匹配可参加的临床研究
